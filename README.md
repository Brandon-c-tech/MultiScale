# MultiScale

A simple, fast tool for generating multiple screenshot sizes for app store submissions. All image processing happens right in your browser - your screenshots never leave your device.

## 🚀 Features

- Drag-and-drop interface for easy image upload
- 100% client-side image processing for privacy
- Support for all major iOS and Android device sizes
- Custom size input option
- Batch processing multiple images
- Export all sizes in a single ZIP file
- Support for @1x, @2x, and @3x scaling

## 🔒 Privacy First

- All image processing happens locally in your browser
- No server uploads required
- Your screenshots never leave your device
- Zero data collection

## 💻 Supported Devices

- iPhone (16 Pro Max to SE)
- Android (Compact & Medium)
- Custom dimensions

## 🛠️ Tech Stack

- React
- TypeScript
- react-dropzone (for file handling)
- JSZip (for packaging resized images)
- Pure front-end processing - no backend needed

## 🚀 Getting Started

1. Visit [MultiScale](https://multiscale.online)
2. Drop your screenshots
3. Select target devices
4. Download your resized images
